# JackPack Native Payloads

This folder is for WebUI-first JackPack payloads.

Legacy RaspyJack payloads stay in their original folders as porting source, but they are not shown in the JackPack runtime until they have been adapted to the native form/workflow model. New payloads created from the WebUI are generated here and can define their own setup fields through `JACKPACK_FORM`.
